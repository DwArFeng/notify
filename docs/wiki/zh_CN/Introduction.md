# notify

Notify 是一个功能全面的通知服务，旨在为用户提供高效、可靠的通知体验。

---

## 特性

- 以 SPI 的形式对通知流程进行标准化，如主题（Topic）、调度（Dispatch）、路由（Route）等。
- 提供 Telqos 运维平台，能够在没有 GUI 的环境下使用本服务的功能。
- 支持主流关系型数据库（基于 Hibernate）。
- 支持分布式部署。
